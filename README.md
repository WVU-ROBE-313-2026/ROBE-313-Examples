# ROBE-313: Robotics Examples

Welcome to the official example code repository for **ROBE-313** at West Virginia University. 

This repository contains the Python scripts, datasets, and interactive examples that accompany the course lectures. These examples are designed to help you bridge the gap between the mathematical theory discussed in class and practical robotics implementation.

## 🚀 Getting Started

Before running these scripts, ensure you have a standard Python 3 environment set up. Many of the vision and matrix operations rely on standard robotics and data science libraries. 

> **Note on ArUco Markers:** > To run the examples (`aruco_pose_image.py` and `aruco_pose_live.py` located in `Module-06/Python-E`), please refer to the step-by-step instructions provided in this week's class slides.
